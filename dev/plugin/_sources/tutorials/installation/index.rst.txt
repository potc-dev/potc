Installation
===================

Potc is currently hosted on PyPI. It required python >= 3.6.

You can simply install potc from PyPI with the following command:

.. code:: shell

    pip install potc

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/potc-dev/potc.git@main

You can check your installation by the following python \
script:

.. literalinclude:: install_check.demo.py
    :language: python
